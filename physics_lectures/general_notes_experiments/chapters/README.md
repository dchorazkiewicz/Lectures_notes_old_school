# Chapters

This project is an incubator for independent physics notes.

Each `chapter_*.tex` file should represent one self-contained idea, model,
thought experiment, derivation, or extended note. A chapter does not need to fit
into a linear course sequence.

Use a clear name such as:

```text
chapter_001_energy_as_a_bookkeeping_principle.tex
chapter_002_why_dimensional_analysis_predicts_scaling.tex
```

If the note grows into course-ready material, it should be easy to copy or adapt
the chapter file and any related figures into `physics_basics` or another
structured course project.
