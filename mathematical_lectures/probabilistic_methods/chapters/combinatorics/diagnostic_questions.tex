\section{Diagnostic questions before counting}

Before using a formula, answer the following questions.

\begin{enumerate}
    \item What is one elementary outcome?
    \item Is the outcome a sequence, a set, an arrangement, or a count vector?
    \item Does order matter?
    \item Are all objects used, or only some of them?
    \item Can the same object be used more than once?
    \item Are the objects distinguishable or indistinguishable?
    \item Are we first counting labelled descriptions and then identifying descriptions that represent the same outcome?
\end{enumerate}

These questions are more important than the names of formulas. The names are only labels attached to structures that have already been recognized.

\begin{remarkbox}
Do not begin a problem by asking which formula you remember. Begin by asking what kind of outcome is being constructed.
\end{remarkbox}
