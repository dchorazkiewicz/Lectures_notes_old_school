\section{Properties visible in small dimensions}

The formulas from the previous section define the determinant. We now investigate what those formulas do. We begin with \(2\times2\) matrices because every calculation can be written explicitly. The same behaviour can also be verified from the \(3\times3\) formula.

\subsection{Linearity in one row}

Let
\[
r=(a,b),\qquad s=(c,d),\qquad u=(p,q).
\]
Using rows as notation, write
\[
\det\begin{pmatrix}r\\s\end{pmatrix}
=\begin{vmatrix}a&b\\c&d\end{vmatrix}=ad-bc.
\]
If the first row is replaced by \(r+u\), then
\[
\begin{aligned}
\det\begin{pmatrix}r+u\\s\end{pmatrix}
&=\begin{vmatrix}a+p&b+q\\c&d\end{vmatrix}\\
&=(a+p)d-(b+q)c\\
&=(ad-bc)+(pd-qc)\\
&=\det\begin{pmatrix}r\\s\end{pmatrix}
+\det\begin{pmatrix}u\\s\end{pmatrix}.
\end{aligned}
\]
Similarly, for a number \(\lambda\),
\[
\det\begin{pmatrix}\lambda r\\s\end{pmatrix}
=\lambda\det\begin{pmatrix}r\\s\end{pmatrix}.
\]

Thus the determinant is linear in the first row when the second row is held fixed. It is also linear in the second row when the first row is held fixed.

\begin{theorembox}
For \(2\times2\) and \(3\times3\) determinants, the determinant is linear in each row separately and in each column separately.
\end{theorembox}

The word \emph{separately} is essential. The determinant is not linear in the whole matrix. For a \(2\times2\) matrix,
\[
\det(\lambda A)=\lambda^2\det A,
\]
because both rows have been multiplied by \(\lambda\).

\subsection{Exchanging two rows}

For a \(2\times2\) matrix,
\[
\det\begin{pmatrix}a&b\\c&d\end{pmatrix}=ad-bc.
\]
After exchanging the rows,
\[
\det\begin{pmatrix}c&d\\a&b\end{pmatrix}=cb-da=-(ad-bc).
\]
Therefore exchanging two rows changes the sign.

\begin{theorembox}
For determinants of size \(2\times2\) and \(3\times3\), exchanging two rows changes the sign of the determinant. The same is true when two columns are exchanged.
\end{theorembox}

This explains the earlier calculation:
\[
\det\begin{pmatrix}1&2\\3&4\end{pmatrix}=-2,
\qquad
\det\begin{pmatrix}3&4\\1&2\end{pmatrix}=2.
\]

\subsection{Equal and proportional rows}

Suppose two rows are equal. Exchanging them does not change the matrix, but exchanging them must change the sign of the determinant. Hence
\[
\det A=-\det A,
\]
which gives
\[
\det A=0.
\]

The same conclusion follows directly from linearity when two rows are proportional. If the second row is \(\lambda r\), then
\[
\det\begin{pmatrix}r\\\lambda r\end{pmatrix}
=\lambda\det\begin{pmatrix}r\\r\end{pmatrix}=0.
\]

\begin{theorembox}
If two rows are equal or proportional, then the determinant is zero. The same statement holds for equal or proportional columns.
\end{theorembox}

For example,
\[
\det\begin{pmatrix}
1&2&3\\
2&4&6\\
0&1&5
\end{pmatrix}=0,
\]
because the second row is twice the first row. There is no need to use Sarrus' rule.

\subsection{Adding a multiple of one row to another}

Let the rows be \(r\) and \(s\). By linearity,
\[
\det\begin{pmatrix}r\\s+\lambda r\end{pmatrix}
=
\det\begin{pmatrix}r\\s\end{pmatrix}
+
\lambda\det\begin{pmatrix}r\\r\end{pmatrix}.
\]
The second determinant is zero, so
\[
\det\begin{pmatrix}r\\s+\lambda r\end{pmatrix}
=
\det\begin{pmatrix}r\\s\end{pmatrix}.
\]

\begin{theorembox}
Adding a multiple of one row to another row does not change the determinant. The same is true for columns.
\end{theorembox}

This property is not a separate trick. It follows from two earlier facts: linearity in one row and the vanishing of a determinant with equal rows.

\begin{examplebox}
Let
\[
A=\begin{pmatrix}2&1\\5&3\end{pmatrix}.
\]
Then
\[
\det A=2\cdot3-1\cdot5=1.
\]
Replace the second row by \(R_2-2R_1\):
\[
\begin{pmatrix}2&1\\5&3\end{pmatrix}
\longrightarrow
\begin{pmatrix}2&1\\1&1\end{pmatrix}.
\]
The new determinant is
\[
2\cdot1-1\cdot1=1,
\]
the same as before.
\end{examplebox}

\subsection{Transposition}

For a \(2\times2\) matrix,
\[
\det\begin{pmatrix}a&b\\c&d\end{pmatrix}=ad-bc
=\det\begin{pmatrix}a&c\\b&d\end{pmatrix}.
\]
The same equality follows from the \(3\times3\) formula.

\begin{theorembox}
For matrices of size \(2\times2\) and \(3\times3\),
\[
\det(A^{\mathsf T})=\det A.
\]
\end{theorembox}

Consequently, every property stated for rows has a corresponding version for columns.

\begin{summarybox}
For the small determinants already defined:
\begin{itemize}
\item the determinant is linear in each row and column separately;
\item exchanging two rows or columns changes the sign;
\item equal or proportional rows or columns give determinant zero;
\item adding a multiple of one row or column to another does not change the determinant;
\item transposition does not change the determinant.
\end{itemize}
\end{summarybox}
