\chapter{Weekly Problem Sets}

This chapter contains one problem set for each substantive chapter of the course. Each set is intended to support one week of study and contains twenty problems arranged approximately from direct practice to interpretation and synthesis.

Unless a problem explicitly asks only for a numerical answer, show the essential reasoning. When a problem asks for an interpretation, explain the meaning of the result in words rather than giving only a formula.
