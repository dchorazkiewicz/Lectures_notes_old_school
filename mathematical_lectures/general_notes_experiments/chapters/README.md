# Chapters

This project is an incubator for independent mathematical notes.

Each `chapter_*.tex` file should represent one self-contained idea, sketch,
experiment, or extended note. A chapter does not need to fit into a linear
course sequence.

Use a clear name such as:

```text
chapter_001_eigenvalues_as_stretching_factors.tex
chapter_002_a_visual_route_to_derivatives.tex
```

If the note grows into course-ready material, it should be easy to copy or adapt
the chapter file and any related figures into `basic_mathematics` or another
structured course project.
