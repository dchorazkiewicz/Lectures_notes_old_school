\chapter{Matrices}

This chapter introduces matrices as organised arrays and as actions on coordinate data. After completing it, the student should be able to:
\begin{itemize}
\item read matrix dimensions, entries, rows, columns, and special structures;
\item perform entrywise operations and matrix multiplication with the correct order;
\item use elementary row operations and recognise why they return in later chapters.
\end{itemize}

\noindent\textbf{Matrices as organised operations.} A matrix has a shape, a positional structure, and a role in a problem. Entrywise operations preserve positions; multiplication combines rows with columns; special patterns reveal simplified behaviour; and row operations expose hidden relations. Row operations will return in the next three chapters with new meanings: tracking determinants, constructing inverses, and preserving solution sets.
