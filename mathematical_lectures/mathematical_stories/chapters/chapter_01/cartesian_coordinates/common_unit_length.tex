\subsection{Constructing a Common Unit Length}

Choose a positive segment length and call it one unit. Using a compass, transfer
this same length from $O$ repeatedly along both axes and in both directions.
The first marks are labelled $1$ and $-1$; repetition constructs the integer
marks
\[
\ldots,-3,-2,-1,0,1,2,3,\ldots
\]
on each axis.

\begin{center}
\begin{tikzpicture}[scale=0.82]
    \draw[axis] (-4.2,0) -- (4.4,0) node[right] {$x$};
    \draw[axis] (0,-4.1) -- (0,4.3) node[above] {$y$};

    % Four successive compass placements.  The same colour denotes the same
    % iteration on the two axes.
    \draw[blue!70,dashed,line width=0.8pt] (0,0) circle (1);
    \draw[orange!85!black,dashed,line width=0.8pt] (1,0) circle (1);
    \draw[orange!85!black,dashed,line width=0.8pt] (0,1) circle (1);
    \draw[green!55!black,dashed,line width=0.8pt] (2,0) circle (1);
    \draw[green!55!black,dashed,line width=0.8pt] (0,2) circle (1);
    \draw[violet!75,dashed,line width=0.8pt] (3,0) circle (1);
    \draw[violet!75,dashed,line width=0.8pt] (0,3) circle (1);

    \foreach \x in {-3,-2,-1,1,2,3} {
        \draw (\x,0.09) -- (\x,-0.09) node[below=3pt] {$\x$};
    }
    \foreach \y in {-3,-2,-1,1,2,3} {
        \draw (0.09,\y) -- (-0.09,\y) node[left=3pt] {$\y$};
    }
    \node[below left] at (0,0) {$0$};

    \fill[blue!70] (0,0) circle (1.8pt);
    \fill[orange!85!black] (1,0) circle (1.8pt);
    \fill[orange!85!black] (0,1) circle (1.8pt);
    \fill[green!55!black] (2,0) circle (1.8pt);
    \fill[green!55!black] (0,2) circle (1.8pt);
    \fill[violet!75] (3,0) circle (1.8pt);
    \fill[violet!75] (0,3) circle (1.8pt);

    \draw[<->,gray] (0,-3.65) -- (1,-3.65);
    \node[gray,below] at (0.5,-3.65) {one common unit};
\end{tikzpicture}
\end{center}

\begin{remarkbox}
The same unit must be used on both axes. A segment labelled as one unit on the
$x$-axis must be congruent to a segment labelled as one unit on the $y$-axis.
This common scale is what later allows the Pythagorean theorem to produce the
standard coordinate formula for Euclidean distance.
\end{remarkbox}
