\subsection{Free Variables and Infinitely Many Solutions}

If a variable column has no pivot, that variable is free and may be used as a parameter.

\begin{examplebox}
The echelon form
\[
\left[\begin{array}{ccc|c}
1&2&-1&3\\0&0&1&4
\end{array}\right]
\]
represents
\[
\begin{cases}
x+2y-z=3,\\z=4.
\end{cases}
\]
Set $y=t$. Then
\[
z=4,
\qquad
x=7-2t.
\]
Thus
\[
\boxed{(x,y,z)=(7-2t,t,4),\quad t\in\mathbb R}.
\]
\end{examplebox}

\begin{interpretationbox}[A parameter records remaining freedom]
The parameter is not an arbitrary decoration. It measures the direction in which the independent conditions fail to determine a unique point.
\end{interpretationbox}
