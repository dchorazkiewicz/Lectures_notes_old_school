\section{Review Exercises}

\subsection{Elementary computational exercises}

\begin{enumerate}
\item Solve by elimination:
\[
\begin{cases}
x+y=7,\\
2x-y=2.
\end{cases}
\]
Write the augmented matrix and verify the solution.

\item Solve by elimination:
\[
\begin{cases}
3x+2y=4,\\
5x-2y=12.
\end{cases}
\]
Choose a step that avoids early fractions.

\item Classify and solve:
\[
\begin{cases}
x+2y=3,\\
2x+4y=6.
\end{cases}
\]
Write the solution set with one parameter.

\item Classify
\[
\begin{cases}
x-3y=2,\\
2x-6y=7.
\end{cases}
\]
and display the contradictory row.

\item Solve
\[
\begin{cases}
2x+y=5,\\
x-y=1
\end{cases}
\]
with Cramer's rule.

\item Solve
\[
\begin{cases}
2x+y=5,\\
x+y=3
\end{cases}
\]
with the inverse-matrix method.

\item For the parameter $a$, classify
\[
\begin{cases}
x+y=3,\\
2x+ay=6.
\end{cases}
\]

\item For the parameter $b$, classify
\[
\begin{cases}
x-2y=1,\\
3x-6y=b.
\end{cases}
\]

\item Find and plot the intersection of
\[
y=2x-3,
\qquad
y=-x+6
\]
by solving the corresponding system.

\item Reduce
\[
\left[\begin{array}{cc|c}
1&2&5\\
3&6&15
\end{array}\right]
\]
to echelon form, identify the pivot and free variable, and write the full solution set.
\end{enumerate}

\subsection{Development and reasoning exercises}

\begin{enumerate}
\item Solve by Gaussian elimination:
\[
\begin{cases}
x+y+z=6,\\
2x-y+z=3,\\
x+2y-z=2.
\end{cases}
\]
State the pivot goal at each stage and verify the final triple.

\item Classify and solve:
\[
\begin{cases}
x+y+z=3,\\
2x+2y+2z=6,\\
x-y+z=1.
\end{cases}
\]
Identify a free variable and explain why the same pivot logic as in $2\times2$ still applies.

\item Classify
\[
\begin{cases}
x+y+z=1,\\
2x+2y+2z=3,\\
x-y=0.
\end{cases}
\]
Show the contradictory row and explain what incompatibility means geometrically.

\item Solve the parameter-dependent system
\[
\begin{cases}
x+y+z=2,\\
x+ay+z=2,\\
x+y+az=2.
\end{cases}
\]
Separate exceptional values before division and classify every case.

\item Solve one invertible $2\times2$ system by elimination, Cramer's rule, and the inverse method. Compare the intermediate objects and explain why all three methods produce the same point.

\item For a singular coefficient matrix, construct one right-hand side giving no solution and another giving infinitely many solutions. Solve both and explain why $\det A=0$ alone does not decide between the two cases.

\item Explain why elementary row operations preserve the solution set. Demonstrate each of the three operation types on one concrete system and compare the equations before and after.

\item Interpret the rows
\[
[\,0\quad0\mid5\,]
\qquad\text{and}\qquad
[\,0\quad0\mid0\,].
\]
Give a system producing each row and describe the corresponding geometry.

\item Compare Gaussian elimination, Cramer's rule, and the inverse-matrix method. Include method scope, classification power, required assumptions, and computational cost.

\item Give a complete strategy for a linear system: translate conditions into an augmented matrix, select pivots, preserve equivalence, detect contradiction or freedom, write the solution set, verify it, and interpret the result geometrically.
\end{enumerate}
