\section{Position and Displacement Vectors}

Coordinates allow us to describe where a point is. Once an origin $O$ has been
chosen, the position of a point $A$ can be represented by the arrow from the
origin to that point. This is the \emph{position vector}
\[
\mathbf r_A=\overrightarrow{OA}.
\]

Motion requires a second use of vectors. If an object moves from $A$ to $B$,
the arrow from the initial position to the final position is the
\emph{displacement vector}
\[
\Delta\mathbf r=\overrightarrow{AB}.
\]
The two ideas are connected by
\[
\boxed{\overrightarrow{AB}=\mathbf r_B-\mathbf r_A}.
\]
Thus position vectors describe where points are relative to the chosen origin,
while displacement vectors describe how one position changes into another.
This distinction will later make it possible to describe motion in physics
without confusing a place with a change of place.

The same pair of components can therefore play two different roles. Written as
$\mathbf r_A$, it locates a point relative to the origin. Written as a
displacement, it is an instruction that can be applied from any starting point.
The operations developed below describe how positions change when
displacements are combined, reversed, rescaled, and compared geometrically.

\noindent\textbf{Vector toolkit.} Position vectors identify points relative to an origin, while displacement vectors describe changes between positions. Addition and scalar multiplication combine and rescale these changes; length and the dot product encode metric information; and parallelism, perpendicularity, angles, projections, and orthogonal decompositions translate geometric questions into coordinate calculations. The same ideas extend from the plane to three-dimensional space.
